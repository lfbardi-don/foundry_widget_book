## Design System

This widget book showcases the [Foundry Design System](https://github.com/lfbardi-don/foundry_ds), featuring:

- **Semantic color tokens** with light/dark theme support
- **Typography scale** following design best practices
- **Spacing & layout tokens** for consistent UI
- **Premium animations** and interactive feedback
